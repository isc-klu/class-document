XData MARKDOWN [ MimeType = "text/markdown" ]
{
# This is some markdown

We can write markdown in here using various *formatting* and other
options:
1. A list
2. Code Blocks
3. Etc.

## Code Block

```json
{ "key": "value", "pi": 3.14159 }
```
}

{
<?xml version="1.0"?>

<root/>
}

}
